Principles
- Bold, minimal, high-contrast. Vibrant purple/cyan accents on deep charcoal.
- Generous whitespace, glass surfaces for focus panels, no decorative boilerplate.
- Clear hierarchy: strong headings, small supportive labels, confident CTAs.

Typography
- Display/Headings: Geist Sans (already bundled) with tighter tracking.
- Body: Geist Sans regular, comfortable line-height.
- Scale: 32/24/18 for H1/H2/H3, 14 body, 12 helper.

Color Tokens (Tailwind via CSS vars)
- Background: hsl(0 0% 4%)
- Foreground: hsl(0 0% 98%)
- Primary: hsl(260 100% 65%)
- Accent: hsl(190 95% 55%)
- Surfaces: card hsl(0 0% 7%), borders hsl(0 0% 16%)

Components
- Buttons: neon gradient (primary), outline on dark for secondary.
- Cards: glassmorphism (blur 12px, white/10 border, deep shadow).
- Inputs: minimal, transparent backgrounds, amber ring replaced by purple ring.

Layout
- Header: simple wordmark + actions; no external brand logos.
- Pages: Home (Hero + CTAs), Create (generator), Gallery (grid), Editor (basic adjustments).
